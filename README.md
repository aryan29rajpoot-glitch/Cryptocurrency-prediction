# cryptocurrency price Forecasting using LSTM

## Project Overview
This project predicts cryptocurrency prices using a Deep Learning LSTM model.

## Features
- Bitcoin historical data visualization
- LSTM-based forecasting
- Actual vs Predicted graph
- Next day Bitcoin price prediction
- Streamlit web app deployment

## Technologies Used
- Python
- TensorFlow/Keras
- Streamlit
- Plotly
- yFinance
- Scikit-learn

## Run Project

### Install dependencies
pip install -r requirements.txt

### Train Model
python model.py

### Run App
streamlit run app.py
